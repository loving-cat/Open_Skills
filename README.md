# Open_Skills
● 专家团模式 (Think Skill)                                                                                        
                                                                                                                  
  启用专家团协作模式，按照"规划→任务分解→专家执行→疑难调度"流程解决复杂开发问题。                                    
  工作流程：                                                                                                      
  1. 架构师 - 输出Plan和任务分解                                                                                  
  2. 专家执行 - 分配给前端/后端/数据库/DevOps专家
  3. 疑难调度 - 自动调度额外专家介入

  触发词： /Think、/专家团、启用专家团、专家团模式

  适用场景： 复杂架构设计、多模块开发、技术选型决策、疑难问题排查

  🤝 专家团协作模式 - 通过"规划→任务分解→专家执行→疑难调度"流程解决复杂开发问题。架构师统筹规划，前端/后端/数据库/
  DevOps专家分工执行，疑难问题自动调度额外专家介入。支持命令 /Think 快速启用。
